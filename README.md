# cn-portal-journal

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件的形式存放，不依赖后端服务，可直接通过浏览器打开或托管于任何静态站点服务。

## 项目简介

`cn-portal-journal` 是一个静态 HTML 页面集合仓库。其目的在于长期保存并公开访问一系列独立的网页内容。每个页面均为自包含的 HTML 文档，可独立呈现，无需特定运行环境或数据库支持。

本仓库不针对任何特定域名、网站或平台，仅作为通用型页面归档与发布载体。

## 目录说明

```
cn-portal-journal/
├── pages/             # 存放所有 HTML 页面文件
│   ├── index.html     # 主入口页（可选）
│   └── ...
├── assets/            # 公共资源（CSS、JS、图片等）
├── docs/              # 相关文档或说明
└── README.md          # 本文件
```

- `pages/`：核心目录，每个 `.html` 文件代表一个独立页面。
- `assets/`：存放页面共用的样式、脚本或图片资源。
- `docs/`：可选，用于放置补充说明或版本记录。

## 页面归档说明

- 所有页面均为纯静态 HTML，不含服务器端脚本。
- 页面之间相互独立，不共享状态或数据。
- 归档内容可能来自不同时期、不同主题，但均以统一格式存放。
- 页面文件名建议使用有意义的英文或拼音，避免特殊字符。

## 维护说明

- 欢迎提交 Pull Request 添加新的独立页面，或修正现有页面中的错误。
- 新增页面请放置在 `pages/` 目录下，并确保其为有效的 HTML 文档。
- 如涉及公共资源更新，请同步修改 `assets/` 中对应文件。
- 本仓库不追踪具体的网站或服务，仅关注页面本身的内容与结构。
